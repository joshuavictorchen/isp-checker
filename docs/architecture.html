Architecture

Framework

Note

Documentation is under construction. This section describes the program at a very high level, for now. This is just a prototype, and the structure here is subject to change.

This program is structured as follows:

  1. User executes the program and provides a formatted, case-insensitive address via the command line

  2. The provided address is parsed into dict of sub-components by Address.parse_address

  3. Address.check_isps instantiates a series of ISP-based objects, which perform unique ISP-specific actions to retrieve metadata (serviceability, supported internet speed, etc.) based on their own execution stacks

  4. Results are printed to the console at runtime

More ISPs can be checked by simply creating new ISP-based classes and adding them to the Address.check_isps function.
